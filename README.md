Emotion-Sensing Affective Robotic System
This project implements a real-time Human-Robot Interaction (HRI) system. It uses Computer Vision and Deep Learning to recognize human emotions via a webcam and translates those emotions into physical robotic movements using an ESP32-controlled 4-wheel chassis.
🚀 System Architecture & WorkflowThe project utilizes a distributed computing approach where a PC handles high-level AI processing and an ESP32 manages real-time hardware actuation.Vision & Detection: OpenCV captures live video, and Haar Cascade classifiers isolate the face (Region of Interest).Emotion Inference: The cropped face is processed by a Mini-Xception model running on TensorFlow.Training: The model is pre-trained on the FER2013 dataset, allowing it to classify 7 emotional states: Happy, Sad, Angry, Surprise, Neutral, Fear, and Disgust.Communication: The PC sends an HTTP GET request wirelessly over Wi-Fi to the ESP32’s local IP address.Actuation: The ESP32 receives the command and signals the L298N Motor Driver to execute specific movements.
